# CodeMirror Editor Support for Markor

## Development

### Install Node.js Module Dependencies

```shell
npm install
```

### Start Development Server

```shell
npm run dev
```

### Build

```shell
npm run build
```

The build output directory is `../app/thirdparty/assets/cm-editor`.
