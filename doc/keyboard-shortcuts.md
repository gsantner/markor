# Keyboard shortcuts for Markor

> Last modified: 2026.02.21

## Common

### Edit

| Shortcut Keys    | Action                                  |
| ---------------- | --------------------------------------- |
| Ctrl + A         | Select all                              |
| Ctrl + C         | Copy                                    |
| Ctrl + X         | Cut current line or selected text       |
| Ctrl + V         | Paste                                   |
| Ctrl + S         | Save                                    |
| Ctrl + Z         | Undo                                    |
| Ctrl + Shift + Z | Redo                                    |
| Ctrl + Shift + R | Reload                                  |
| Ctrl + Shift + U | Change case                             |
| Ctrl + D         | Duplicate current line or selected text |
| Ctrl + G         | Go to a specific line                   |
| Ctrl + K         | Delete from cursor to the end of line   |
| Ctrl + Alt + K   | Delete current line                     |
| Ctrl + W         | Select word                             |
| Ctrl + L         | Select current line                     |
| Ctrl + Alt + L   | Select a range of lines                 |
| Shift + ↑ ↓ ← →  | Select text from cursor                 |
| Ctrl + Alt + ↑   | Duplicate line up                       |
| Ctrl + Alt + ↓   | Duplicate line down                     |
| Ctrl + ↑         | Move line up                            |
| Ctrl + ↓         | Move line down                          |
| Ctrl + ←         | Move to word start                      |
| Ctrl + →         | Move to word end                        |
| Alt + ←          | Move to line start                      |
| Alt + →          | Move to line end                        |
| Alt + ↑          | Move to document start                  |
| Alt + ↓          | Move to document end                    |
| Home             | Move to line start                      |
| End              | Move to line end                        |
| Page Up          | Page up                                 |
| Page Down        | Page down                               |
| Shift + Enter    | Insert new line below                   |
| Ctrl + .         | Let the editor get focus                |
| Ctrl + ,         | Show special key dialog                 |

### Edit & View

| Shortcut Keys | Action                                           |
| ------------- | ------------------------------------------------ |
| Ctrl + F      | Search                                           |
| Ctrl + P      | Print                                            |
| Ctrl + /      | Toggle to preview/edit                           |
| Ctrl + ;      | Show more menu options                           |
| Ctrl + '      | Run title click (to show table of contents, ...) |
